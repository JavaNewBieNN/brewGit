Python 容器(Containers)是什么?
在 Python 中,容器(Container) 是一种可以存储多个元素的数据结构,它们允许我们存储、访问、操作和遍历数据.
容器可以存放不同类型的对象,甚至可以包含其他容器.

---------------------------------------------------------------------------------------------------------
Python 中常见的容器类型
Python 的标准容器主要包括以下几种:

容器类型	特点	可变性                                                          特点
列表(list)	有序、支持索引、支持重复元素	                        ✅ 可变             万能
元组(tuple)	有序、支持索引、支持重复元素	                        ❌ 不可变           不可变的list
集合(set)	无序、不支持索引、元素唯一	                            ✅ 可变             元素唯一,无序,不支持索引
字典(dict)	无序(Python 3.6+ 以后是有序的),键唯一	               ✅ 可变              
deque(双端队列)	collections.deque,比 list 适合做队列操作	       ✅ 可变

----------------------------------------------------------------------------------------------------------

切片:从一个序列中取出一个字序列
语法:序列[起始下标:结束下标:步长]

步长 1 表示一个个取元素
步长 2 表示间隔一个取元素
步长 N 表示每次跳过 N-1个元素取
步长为负数表示反向取
